Bem-vindo à API de Agendamento de Tarefas

Rotas Disponíveis:

  • GET /tarefas - Retorna todas as tarefas
  • GET /tarefas/<id> - Retorna a tarefa pelo ID
  • POST /add - Adiciona uma nova tarefa
  • PUT /tarefas/<id> - Atualiza uma tarefa pelo ID
  • PATCH /tarefas/<id> - Atualiza uma das informações da tarefa pelo ID
  • DELETE /tarefas/<id> - Remove uma tarefa pelo ID
  • GET /restart - Reinicia os dados
